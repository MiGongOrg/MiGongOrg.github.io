JavaScript eval() 安全隐患

Posted on 2016-06-15   |   In JavaScript

为什么 eval() 存在安全隐患

eval() 函数可以将字符进行编译后执行，如果使用 eval() 处理 JSON 数据，当服务器本身或发来的 JSON 数据被劫持，就很容易被利用，在客户端编译执行恶意代码。

如何防范和避免

  1. 使用 JSON.parse() 代替 eval() 因为 JSON.parse() 更加安全
  2. 将 JSON 数据中的 html 字符进行转码
