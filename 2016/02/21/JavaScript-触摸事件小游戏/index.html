JavaScript 触摸事件小游戏

Posted on 2016-02-21   |   In JavaScript / JQuery

CSS Code: 禁用移动端按住选取文本

-webkit-user-select: none; /* 禁用选取文本 */
/* pointer-events: none; 另一种方法 */

JavaScript Code:

触摸开始 touchstart：监听触摸开始事件，当按住时触发相应函数。

addEventListener("touchstart",function(){...},false);

触摸结束 touchend：监听触摸结束事件，当触摸结束时触发相应函数。

addEventListener("touchend",function(){...},false);

触摸移动 touchmove：禁用触摸移动事件，避免上下或左右滑动。

document.addEventListener("touchmove", function(e){   //禁用触摸移动
    e.preventDefault()    //阻止默认行为
});
