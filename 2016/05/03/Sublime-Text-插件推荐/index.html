Sublime Text 插件推荐

Posted on 2016-05-03   |   In Sublime Text

Sublime Text 必装插件

Emmet 快速编写 HTML与CSS 就不多做介绍了，是 Sublime 必装插件之一，具体使用方法和快捷键介绍请查看 Sublime Text 插件 Emmet 常用快捷键

Emmet 插件地址：https://packagecontrol.io/packages/Emmet

Sublime​Code​Intel 代码提示，自动补全

Sublime​Code​Intel 插件地址：https://packagecontrol.io/packages/SublimeCodeIntel

Sublime Text 命令行（Shell）相关插件和可选插件

Terminal 在编辑文件时打开 Cmder 默认是当前文件所在目录

快捷键：Ctrl+Shift+T

配置方法详见：http://www.jianshu.com/p/20eb268b2223

插件地址：https://packagecontrol.io/packages/Terminal

Shell Turtlestein 在 Sublime 中执行 shell 命令

Shell Turtlestein 快捷键：

Ctrl + Shift + C      //打开执行命令行
Ctrl + Alt+Shift + C  //打开CMD窗口
Ctrl + Shift + X      //重新运行上一次命令

Shell Turtlestein 插件地址：https://packagecontrol.io/packages/Shell%20Turtlestein

View In Browser 使用浏览器打开当前文件 .html 文件

快捷键：自定义后，使用不同快捷键可以打开相应浏览器

设置方法：Preferences > Key Bindding - User

[
    { "keys": [ "ctrl+alt+v" ], "command": "view_in_browser" },
    { "keys": [ "ctrl+alt+f" ], "command": "view_in_browser", "args": { "browser": "firefox" } },
    { "keys": [ "ctrl+alt+c" ], "command": "view_in_browser", "args": { "browser": "chrome" } },
    { "keys": [ "ctrl+alt+i" ], "command": "view_in_browser", "args": { "browser": "iexplore" } },
    { "keys": [ "ctrl+alt+s" ], "command": "view_in_browser", "args": { "browser": "safari" } }
]

通过相应快捷键，打开相应浏览器

CTRL + ALT + F - Firefox
CTRL + ALT + C - Chrome
CTRL + ALT + I - Internet Explorer
CTRL + ALT + S - Safari

View In Browser 插件地址：https://packagecontrol.io/packages/View%20In%20Browser

SublimeLinter 代码检测插件

SublimeLinter 插件地址：https://packagecontrol.io/packages/SublimeLinter

SFTP 是一款付费的 Sublime 文件同步插件，通过配置 SFTP 你可以在本地编辑服务器上的文件 Ctrl+S 保存后会自动上传，还可以同步某个项目文件夹，方便整个项目的修改与更新。

Sublime SFTP 插件地址:https://packagecontrol.io/packages/SFTP

Sublime SFTP 购买地址：https://wbond.net/sublime_packages/sftp/buy

Sublime Text 样式与配色

Alignment 代码样式对齐，专为有洁癖的你而准备的。

快捷键：Ctrl+Shift+A （因为与QQ截图快捷键冲突，所以需要自定义）

自定义快捷键 Preferences > Package Settings > Alignment > Key Bindding - User

[{ "keys": ["ctrl+alt+q"], "command": "alignment" }]

Alignment 插件地址：https://packagecontrol.io/packages/Alignment

SyncedSidebrBg 侧边栏目录颜色与主题配色同步，安装后需要重启 Sublime Text

Sublime Text 主题配色方案：http://colorsublime.com/
