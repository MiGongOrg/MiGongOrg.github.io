JavaScript 判断移动端横屏或竖屏

Posted on 2016-01-12   |   In JavaScript / JQuery

JavaScript 判断移动端横屏或竖屏 Code :

通过监听 window.onorientationchange 是否有改变来判断移动端设备横屏或竖屏，其中 window.orientation 可以判断相应角度，如：

竖屏：window.orientation == 180 || window.orientation == 0
横屏：window.orientation == 90 || window.orientation == -90

window.onorientationchange = function(){
    if(window.orientation == 0){    //简单的判断方法
        alert("竖屏");
    } else {
        alert("横屏");
    }
};

如果部分移动端设备不支持 onorientationchange 事件，也可以使用 resize

var evt = "onorientationchange" in window ? "orientationchange" : "resize";
window.addEventListener(evt, function() {
    alert("屏幕旋转");
}, false);
