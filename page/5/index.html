WAMP配置本地任意域名访问不同文件夹

Posted on 2015-05-09   |   In PHP

修改：wamp\bin\apache\Apache2.4.4\conf\httpd.conf
onlineoffline tag - don’t remove 与 </Directory> 中间内容中的

     #Deny from all //添加#注释

#Allow from 127.0.0.1 //添加#注释
Allow from all //新增


去除：#Include conf/extra/httpd-vhosts.conf 的注释 “#”
改为：Include conf/extra/httpd-vhosts.conf
修改：wamp\bin\apache\Apache2.4.4\conf\extra\httpd-vhosts.conf

在底部添加或修改：

<VirtualHost :80>
    ServerAdmin admin@wp.com
    DocumentRoot “e:/wmap/wamp/www/wp”  //wp 建文件夹
    ServerName www.wp.com          //任意域名
    ErrorLog “logs/www.wp.com-error.log”
    CustomLog “logs/www.wp.com-access.log” common
</VirtualHost>

*修改：C:\Windows\System32\drivers\etc\hosts

在hosts中添加：127.0.0.1 www.wp.com

测试：
1.在wamp\www\wp 中新建 1.php
2.左键点击WAMP右下角图标重启WAMP所有服务
3.浏览器访问http://www.wp.com/1.php

阿里云Linux云服务器数据盘挂载方法

Posted on 2014-11-04   |   In Linux

前几天有申请一个免费试用半年的阿里云服务器，这几天计划把放在国外的那几个网站搬回国内。所以写这篇文章记录一下自己的操作流程，避免以后配置服务器的时候还要到处去找相关教程。在上一篇关于“阿里云服务器Linux操作系统配置部署”中并没有提到数据盘挂载的内容，而且选择了WDCP。今天的这个教程包括：“数据盘挂载”和 安装AMH面板。

直接进入主题，当你成功登陆服务器后，1.输入命令：fdisk -l 检测是否有数据盘，也就是在你购买服务器选择的时候，要添加一块数据盘，而不是赠送的那个系统盘。（其中的 df -h 命令在没有分区和格式化数据盘之前是无法看到数据盘的，所以不用理会，直接输入fdisk -l 即可）

2.将数据盘分区，执行命令 fdisk /dev/xvdb 根据下图红线提示依次输入“n”，“p”“1”，两次回车，“wq”，即可。

3.使用命令 fdisk -l 查看新分区，如下图（与红线部分一致）。

4.使用命令 mkfs.ext3 /dev/xvdb1 格式化分区，如下图所示。

5-1.使用命令 echo ‘/dev/xvdb1 /mnt ext3 defaults 0 0’ >> /etc/fstab 添加分区信息。
5-2.使用命令 cat /etc/fstab 查看分区信息是否写入成功。（与下图相同，表示成功）

6.使用命令 mount -a 挂载新分区，然后用 df -h 命令查看，出现如下图所示信息就说明挂载成功，可以开始使用新的分区了。

以上操作一共6步，每一步的操作对照示例图片进行即可。

原文请参考官方教程：http://help.aliyun.com/view/11108189_13435365.html?spm=5176.7224609.1997282825.5.B85kic

接下来是安装AMH面板：

官网地址（版本选择）：http://amh.sh/AMH/doc.htm

按照提示，执行 AMH 安装脚本即可，非常简单只需要几步。友情提示：不要忘记你输入的SQL数据库密码和AMH登陆密码。

Wampserver环境下修改MySql数据库初始“空”密码

Posted on 2014-05-23   |   In PHP

在Wampserver集成开发环境V2.4中有提到，安装初始Wampserver环境的时候Mysql数据库的初始密码是“空”，所以我们要修改一下密码，但是很多初学者不知道如何修改Mysql数据库的初始密码，接下来你可以按照以下的操作步骤去修改。

其实修改MySql数据库的方法很简单：

1.左键单击Wampserver右下角图标，打开Mysql控制台。

2.进入控制台后会看到Enter Password,它是让你输入密码，因为密码是“空”所有直接回车即可。

3.回车之后输入以下Mysql命令：

mysql> use mysql          
Database changed        
mysql>
4.然后输入要修改的密码的Sql语句之后回车： 
update user set password=PASSWORD('abcdef') where user='root';
5.输入以上命令之后回车，在输入下面的Sql语句后回车： 
flush privileges;
注意：如果不输入最后这一段执行命令，那么修改密码的操作将不会生效。 6.现在可以退出控制台，因为已经修改完成！你可以登录http://127.0.0.1/phpmyadmin/或http://localhost/phpmyadmin/进行测试。 PS:我到这6步已经修改成功了，如果你那还是出错，那么还需要进行第7步骤的操作。 7.修改phpMyAdmin的配置文件，打开文件wamp/apps/phpmyadmin/config.inc.php修改下面这行代码中的密码。 
$cfg['Servers'][$i]['password'] = '你的密码';

PHP初学者必备Wampserver集成开发环境V2.4

Posted on 2014-05-23   |   In PHP

刚接触PHP的时候需要一个集成环境，Wampserver可以帮你解决这一的问题，Wampserver其中包括：Apache、Mysql、PHP三种软件，只要下载它一键安装即可。如果你的电脑安装的不是在Windos系统那么就先选择其它集成环境安装包，因为Wampserver中的Wamp意思就是在Windos下的集成软件安装。Wampserver集成环境V2.4所支持的环境有：Win7/Vista/WinXP/Win2003这四种操作系统。

一、下载安装：（文章最下方提供Wampserver集成环境V2.4下载）

解压后安装Wampserver2.4-x86.exe按照提示下一步，下一步这样操作就可以了，初始界面是英文的，当你安装成功之后屏幕右下角会出现Wampserver图标，右键点击选择Language修改语言即可。

二、相关介绍：

1.在你安装路径下会获得wamp这样的一个文件夹，打开之后你会看到www的文件夹，这就是网站程序所有放置的目录，也可以在这个文件夹下新建一个类似于Wordpress这样的文件夹，当你在默认浏览器访问地址输入：http://127.0.0.1/wordpress/或http://localhost/wordpress/时即可访问该文件夹下的网站程序。

2.MySql数据库你可以直接在浏览器中输入http://127.0.0.1/phpmyadmin/或http://localhost/phpmyadmin/即可访问，默认用户名是root而密码则是“空”。如果咱们拿Wordpress举例，在你安装Wordpress博客平台之前，你需要在MySql数据库中新建一个数据库，你可以随意命名，点击数据库-》输入数据库名-》新建即可。但是这里有一个问题，初始Wampserver环境下的MySql数据库密码是空的，那又如何修改MySql数据库的初始密码那？请前往查看：[Wampserver环境下修改MySql数据库初始“空”密码](http://www.migong.org/?p=650)

3.成功新建数据库和修改数据库密码之后就可以访问http://127.0.0.1/wordpress/或http://localhost/wordpress/安装Wordpress博客平台了，按照提示填写相应的数据信息即可，这一切其实很简单，完成以上步骤你就已经搭建好了一个基于PHP测试环境的Wordpress博客平台。

阿里云服务器Linux操作系统配置部署

Posted on 2014-05-17   |   In Linux

在购买阿里云服务器的时候选择的操作系统是CentOS，众所周知CentOS是Linux发行版之一，至于为什么选择CentOS或者CentOS到底好在哪里，其实我并不能给出你明确的答案，因为我也是个服务器方面的小白。但是我厚着脸皮到处问，花了一天时间去逛论坛、找心得、看教程。最后决定使用CentOS操作系统。

最低配置，如下图：

 

购买成功之后开始配置服务器，对于小白来说果断选择安装可视化面板来进行操作，可以帮助你节省大量时间，但是问题又来了，我又该选择安装什么样的Linux虚拟主机管理系统那？别急，我帮你分析一下。

通过查找资料、逛论坛、看博客我总结了以下国内常用的几款Linux虚拟主机管理操作面板：

  • 1.WDCP 官网：http://www.wdlinux.cn/wdcp/
  • 2.主机宝 官网：http://z.admin5.com/
  • 3.AMH 官网：http://amysql.com/AMH.htm
  • 4.LuManager 官网：https://www.zijidelu.com/

    WDCP：是这四款里面功能比较强大的，所有的方方面面你能操作到的，在WDCP面板中都能找到，但资源占用会高，而且不太适合新手。

    主机宝：是与阿里云合作的一款主机面板，操作比较简单，容易上手，功能虽然不是很多，但是也够小白用了。

    AMH：相对来说AMH非常简洁清爽，操作简单，资源占用比主机宝高，但比WDCP低。而且使用难度会比主机宝高一些。

    LuManager：这是我最早用过的一款，功能与WDCP不相上下，如果与主机宝和AMH对比起来，同样也不太适合新手。
    最后我还是果断的选择了WDCP虚拟主机管理系统，本打算使用LuManager来着，但是想尝试一下新东西所以这次就选择了WDCP。

一、安装配置方法：

1.首先下载Putty（在360软件管家中就能找到），之后输入IP地址，选择SSH类型，端口是22，之后Open打开输入用户名和密码回车后连接（用户名和密码是购买阿里云服务器后系统自动发送到你手机和邮箱中的，其中在你输入密码的时候是不显示光标和动态的）。

2.选择WDCP的RPM包安装，按照以下说明输入即可。

wget http://down.wdlinux.cn/in/lanmp_wdcp_ins.sh（回车）
sh lanmp_wdcp_ins.sh（回车）

执行这样的命令之后，服务器会自动下载RPM包进行安装，你只要等待显示安装成功即可。（这个过程只需要短短的几分钟）

二、使用测试说明：

1.按照上面操作完成WDCP安装后，打开浏览器输入：http://你的ip:8080

默认的用户名:admin 默认的密码:wdlinux.cn
Mysql默认的用户名:root 默认的密码:wdlinux.cn

2.成功进入WDCP管理面板后，首先要修改默认密码，以及创建测试网站。如果你已经操作到这一步，相信对于可视化面板操作就不会有任何问题了，按照面板中所提供的功能进行操作，哪里不太懂的就搜索一下相关教程，面板的使用介绍我就不在此多说了。
