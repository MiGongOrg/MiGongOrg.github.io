JavaScript 简易 Tab 选项卡

Posted on 2015-07-22   |   In JavaScript / JQuery

学习笔记：

this 触发当前操作事件

oButton[i].index = i; 给每个button添加一个自定义属性 index

<button>按钮</button>
如果在 Html 中直接添加自定义的 index 属性会被浏览器过滤掉，所以使用 Javascript [![tab](http://www.migong.org/wp-content/uploads/2015/07/tab.png)](http://www.migong.org/wp-content/uploads/2015/07/tab.png) **Html 代码：** 因测试 Javascript 代码，所以 Html 部分做了简化，使用的切换标签是 Button 
<button class="thisSylte">最新</button>
<button>热门</button>
<button>随机</button>
<button>评论</button>
<div style="display:block">最新内容</div>
<div>热门内容</div>
<div>随机内容</div>
<div>评论内容</div>
**JavaScript 代码：** 
window.onload = function (){
    window.onload = function () {
    var oButton = document.getElementsByTagName('button');
    var oDiv = document.getElementsByTagName('div');
    for (var i =  0; i < oButton.length; i++) {
        oButton[i].index = i; //给每个button添加一个自定义属性index
        oButton[i].onclick = function () {
            //for循环清空未被选中的button与div样式
            for (var i = 0; i < oButton.length; i++) {
                oButton[i].className = '';
                oDiv[i].style.display = '';
            };
            //点击button后给当前（this）添加样式
            oButton[this.index].className = 'thisSylte';
            oDiv[this.index].style.display = 'block';
            };
        };
    };

JavaScript Checkbox (全选/不选/反选)

Posted on 2015-07-21   |   In JavaScript / JQuery

学习笔记：

window.onload = function () {}文档加载完成后执行 function 内代码

checked=true 与 checked=false 设置 checkbox 选中状态

getElementById 与 getElementsByTagName 的区别

if(){}else{} 判断 与 for(){} 循环实现反选

Html代码：

<button id="quanxuan">全选</button>
<button id="buxuan">不选</button>
<button id="fanxuan">反选</button>
<input type="checkbox" />
<input type="checkbox" />
<input type="checkbox" />
...
**JavaScript 代码：** 
window.onload = function () {
    var quanxuan = document.getElementById('quanxuan')
    var buxuan = document.getElementById('buxuan')
    var fanxuan = document.getElementById('fanxuan')
    var box = document.getElementsByTagName('input')

    quanxuan.onclick = function () {
        for (var i =  0; i < box.length; i++) {
            box[i].checked=true
        };
    }

    buxuan.onclick = function () {
        for (var i =  0; i < box.length; i++) {
            box[i].checked=false
        };
    }

    fanxuan.onclick = function(){
        for (var i =  0; i < box.length; i++) {
            if (box[i].checked == true) {
                box[i].checked=false
            } else{
                box[i].checked=true
            };

        };

    } 

}

Linux 命令解压RAR文件方法

Posted on 2015-06-19   |   In Linux

前言：因本人使用的是AMH主机面板，所以在上传大文件后通过面板后台的AMFTP智能解压时出错无法解压，所以只能通过Linux的命令解压。因上传的是RAR文件，所以需要事先安装unrar在线解压缩软件，我使用的是阿里云的云主机，安装的CentOS 6.3 64位，所以首先要下载相应的unrar在线解压缩软件。

我选择的是：http://www.rarsoft.com/download.htm 中的 RAR 5.21 for Linux

执行以下命令：

wget http://www.rarsoft.com/rar/rarlinux-5.2.1.tar.gz
tar -zxvf rarlinux-3.9.3.tar.gz
cd rar
make

依次执行命令后会显示如下：（表示成功安装）

mkdir -p /usr/local/bin
mkdir -p /usr/local/lib
cp rar unrar /usr/local/bin
cp rarfiles.lst /etc
cp default.sfx /usr/local/lib

执行解压命令：

nurar x content.rar //将content.rar解压到当前文件夹中
示例：将/home/wwwroot/beiying/web 下的 content.rar 解压到 /home/wwwroot/beiying/web 中
nurar x /home/wwwroot/beiying/web/content.rar /home/wwwroot/beiying/web

注意：

nurar e content.rar //将content.rar解压到当前文件夹下， 不按照目录结构。

如果你要解压的文件是分目录结构的，请不要使用该命令。

部分可能用到的命令：

ls -a  //查看当前目录下的所有文件
rm a.txt  //删除单个文件
rm -r admin/  //删除admin文件夹,如果该文件夹下有文件，会每个都提示你是否删除。

如果想直接删除admin文件夹下的所有文件，可以直接使用：

rm -rf admin  // 强制删除admin目录

Sublime Text 插件 Emmet 常用快捷键

Posted on 2015-06-10   |   In Sublime Text

Emmet 是 Sublime Text 快速编辑 HTML CSS 的必备插件，以下记录一些常用快捷方式以备查询。

HTML:

Lorem 排版文字批量生成：

lorem 按Tab后生成排版随机单词字符（如下图）

Lorem 扩展使用方法：

lorem100 按Tab后生成100个随机排版单词字符
div&gt;lorem 按Tab后在div中生成随机排版单词字符，可以与其它HTML标签配合使用，如果p,h1,span等等。

快捷键获取img图片大小：

<img src="lorem.png" alt="" />

按Ctrl+U键，可获取当前img图片的大小。注释：光标必须停留在src="lorem.png"的双引号内在按快捷键，此快捷键在CSS文件中同样可用。

<img src="lorem.png" alt="" width="1201" height="496" />

CSS:

基础快速编辑方式：可无限延伸，请查看此文底部扩展链接。

W100 按Tab后生成 width: 100px; 注释：默认值是px
W100p 按Tab后生成 width: 100%; 注释：P的默认值是%，类似值还可延伸 e 或 x

生成多个属性值：
m20-30 按Tab后生成 margin: 20px 30px; 注释：默认px值需要用-分割
m20em30em 按Tab后生成 margin: 20em 30em; 注释：添加其它单位值，无需使用-连接

另外两种情况演示：

m20--30 按Tab后生成 margin: 20px -30px;

m20em-30em 按Tab后生成 margin: 20em -30em;
一次性生成多条语句，使用“+”号连接：

h10+m20 按Tab后生成 height: 10px; margin: 20px;

快速生成颜色值：

c#f 按Tab后生成 color: #fff;
c#f0 按Tab后生成 color: #f0f0f0;

生成边框颜色简写组合：

bd1#fs 按Tab后生成 border: 1px #fff solid;

生成font-face结构：

@f 按Tab后生成

@font-face {
    font-family:;
    src:url();
}

生成增强型的font-face结构：

@f+ 按Tab后生成

@font-face {
    font-family: ‘FontName’;
    src: url(‘FileName.eot’);
    src: url(‘FileName.eot?#iefix’) format(‘embedded-opentype’),
         url(‘FileName.woff’) format(‘woff’),
         url(‘FileName.ttf’) format(‘truetype’),
         url(‘FileName.svg#FontName’) format(‘svg’);
    font-style: normal;
    font-weight: normal;
}

CSS3浏览器兼容前缀批量生成：

-transform 按Tab后将同时生成 注释：CSS3单词属性前加-

-webkit-transform: ;
-moz-transform: ;
-ms-transform: ;
-o-transform: ;
transform: ;

线性渐变背景生成：

lg(left top,#fff,#000) 按Tab后将同时生成

background-image: -webkit-gradient(linear, 0 0, 100% 100%, from(#fff), to(#000));
background-image: -webkit-linear-gradient(left top, #fff, #000);
background-image: -moz-linear-gradient(left top, #fff, #000);
background-image: -o-linear-gradient(left top, #fff, #000);
background-image: linear-gradient(left top, #fff, #000);

注释：线性渐变快捷方式的（）中有3个属性值，第一个属性是位置，如果填写top,那就是从上↑到下↓，如果是left top，那就是从左上↖到右下↘，以此类推。后面的2个属性是颜色，也就是渐变的开始颜色与结束颜色。

示例：如果想批量修改下面代码中的颜色属性值，只需要修改一个后，按Shift+Ctrl+R即可批量修改其余4个相同属性值。

background-image: -moz-linear-gradient(left top, #fff, #000);
background-image: -o-linear-gradient(left top, #fff, #000);
background-image: linear-gradient(left top, #fff, #000);

Sublime Text 设置快速启动方式

Posted on 2015-06-09   |   In Sublime Text

  1. 将桌面 Sublime Text 快捷方式重命名为 sub
  2. 复制重命名后的 Sublime Text 快捷方
  3. 在F盘(任意盘)新建文件夹 sub（任意文件夹名） ,将重命名后的 Sublime Text 快捷方 sub 粘贴到该文件夹中
  4. 复制该文件夹路径，F:\sub
  5. 右键我的电脑->属性->高级系统设置->环境变量
  6. 编辑变量PATH，在变量值的末尾添加“;”分号，粘贴路径：F:\sub 后保存

    运行快捷键 Win+R 输入sub Enter后即可打开 Sublime Text

其它已安装软件设置快速启动方式步骤相同，系统默认常用快速启动方式还有：（个人）

notepad - 打开记事本
calc - 打开计算器

如需了解更多，请在搜索引擎查找关键词：运行命令大全
