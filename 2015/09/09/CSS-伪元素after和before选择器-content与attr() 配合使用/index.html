CSS 伪元素 :after 和 :before 选择器 content 与 attr() 配合使用

Posted on 2015-09-09   |   In HTML / CSS

CSS 伪元素 :after 和 :before 选择器 content 与 attr() 配合使用都能做些什么？

[](http://www.migong.org "Hello")

取属性值并输出：

div a:after{
    content: attr(data-line);  /* 取自定义属性值，输出结果：Hello World */
    content: attr(title);  /* 取 title 属性值，输出结果：Hello */
    content: attr(alt);  /* 取 alt 属性值，输出结果：World */
    content: attr(href);  /* 取 href 属性值，输出结果：http://www.migong.org */
}

取属性值拼接后输出：

div a:after{
    content: "["attr(data-line)"!]";  /* 字符串拼接，输出结果：[Hello World!] */
}

CSS Code:

/* CSS 伪元素 :after 和 :before 选择器 content 与 attr() 配合使用实例 Style */
.test{
    margin: 0 auto;
    width: 150px;
    height: 50px;
    text-align: center;
    margin-top: 10%;
}
.test a {
    padding: 10px 0;
    margin: 0 20px;
    color: #797878;
    text-shadow: none;
    position: relative;
    text-decoration: none;
    text-transform: uppercase;
    letter-spacing: 1px;
    display: inline-block;
    outline: medium none;
}
.test a::before {
    position: absolute;
    top: 0;
    left: 0;
    overflow: hidden;
    padding: 10px 0;
    max-width: 0;
    border-bottom: 2px solid tomato;
    color: tomato;
    content: attr(data-test);
    -webkit-transition: max-width 0.5s;
    -moz-transition: max-width 0.5s;
    transition: max-width 0.5s;
}
.test a:hover::before {
    max-width: 100%;
}
