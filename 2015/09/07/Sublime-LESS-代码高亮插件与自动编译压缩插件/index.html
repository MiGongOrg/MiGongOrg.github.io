Sublime LESS 代码高亮插件与自动编译压缩插件

Posted on 2015-09-07   |   In Sublime Text

LESS 代码高亮插件：

使用方法：下载 LESS 代码高亮插件，解压后将文件夹放入相应路径（相应的配色方案也可以放在 LESS 代码高亮插件文件夹中，后缀命名为.tmTheme，可在首选项 > 主题方案 > LESS-sublime-master 中启用）

  • OS X: ~/Library/Application Support/Sublime Text 2/Packages/
  • Linux: ~/.Sublime Text 2/Packages/
  • Windows: %APPDATA%/Sublime Text 2/Packages/

插件设置：Preferences > Package Settings > LESS （首选项 > 插件设置 > LESS），将下面代码添加到 Settings User 中保存，将 false 修改成 true 会自动添加 :;

{
  // Boolean setting to auto-insert a semicolon after a ":" is typed.
  "auto_insert_semicolon": false
}

以上步骤完成之后，重启 Sublime 新建 .less 即可看到高亮效果

LESS 代码高亮插件下载地址：Package Control (LESS) | Github (LESS)

LESS 自动编译压缩插件：

使用方法：下载 LESS 自动编译压缩插件，解压后将文件夹放入相应路径

  • Windows: %APPDATA%/Sublime Text 2/Packages/

使用方法与设置：

  • 使用时，每保存*.less 文件时会自动编译生成 同名的.css文件。
  • 在菜单 Preferences>Package Settings>Lessc>Settings 中可以设置是否压缩编译后的css（默认是 false修改成 true 后会自动压缩 ）
  • 不想使用该插件 可以禁用此包（Package Control:Disable Package选择lessc）也可直接删除lessc文件夹卸载插件

LESS 自动编译压缩插件（目前只支持 windows 平台）下载地址：Github (lessc)
