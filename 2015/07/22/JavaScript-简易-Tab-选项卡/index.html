JavaScript 简易 Tab 选项卡

Posted on 2015-07-22   |   In JavaScript / JQuery

学习笔记：

this 触发当前操作事件

oButton[i].index = i; 给每个button添加一个自定义属性 index

<button>按钮</button>
如果在 Html 中直接添加自定义的 index 属性会被浏览器过滤掉，所以使用 Javascript [![tab](http://www.migong.org/wp-content/uploads/2015/07/tab.png)](http://www.migong.org/wp-content/uploads/2015/07/tab.png) **Html 代码：** 因测试 Javascript 代码，所以 Html 部分做了简化，使用的切换标签是 Button 
<button class="thisSylte">最新</button>
<button>热门</button>
<button>随机</button>
<button>评论</button>
<div style="display:block">最新内容</div>
<div>热门内容</div>
<div>随机内容</div>
<div>评论内容</div>
**JavaScript 代码：** 
window.onload = function (){
    window.onload = function () {
    var oButton = document.getElementsByTagName('button');
    var oDiv = document.getElementsByTagName('div');
    for (var i =  0; i < oButton.length; i++) {
        oButton[i].index = i; //给每个button添加一个自定义属性index
        oButton[i].onclick = function () {
            //for循环清空未被选中的button与div样式
            for (var i = 0; i < oButton.length; i++) {
                oButton[i].className = '';
                oDiv[i].style.display = '';
            };
            //点击button后给当前（this）添加样式
            oButton[this.index].className = 'thisSylte';
            oDiv[this.index].style.display = 'block';
            };
        };
    };
