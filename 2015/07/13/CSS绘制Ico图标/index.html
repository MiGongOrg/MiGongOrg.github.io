CSS 绘制 Ico 图标

Posted on 2015-07-13   |   In HTML / CSS

尝试用CSS绘制Ico图标，这组图标共12个，因为时间关系只实现了5个，剩余7个图标相对会难一些，但还是可以用CSS来实现的。

主要用到的CSS3属性：

border-radius: 5px;
box-shadow: 3px 2px 3px 0px rgba(0,0,0,0.3);
transform: rotate(-50deg);

其中有部分相似图形都可以用box-shadow的重复阴影效果来实现。
