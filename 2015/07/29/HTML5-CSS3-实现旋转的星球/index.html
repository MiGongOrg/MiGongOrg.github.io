HTML5 / CSS3 实现旋转的星球

Posted on 2015-07-29   |   In HTML / CSS

主要 HTML5 代码片段：&lt;input max="10" min="1" type="range" /&gt;

主要 CSS3 代码片段：

animation:myfirst 8s infinite; /*infinite 无限循环，但是有停顿*/
animation-timing-function: linear; /*无限循环，无停顿，动画从头到尾的速度是相同的*/

@keyframes myfirst
{
0% {transform: rotate(0deg);}
100% {transform: rotate(360deg);}
}

JavaScript Code:

function goTime(){
    var oInput = document.getElementsByTagName('input')[0];
    var oEarth = document.getElementById('earth');
    var oTimeValue = document.getElementById('timeValue');

    oInputValue = oInput.value;
    oEarth.style.animation = oInputValue + 's infinite linear myfirst';
    oEarth.style['-webkit-animation'] = oInputValue + 's infinite linear myfirst';  // 解决webkit兼容问题
    oTimeValue.innerHTML = oInputValue;
}
setInterval(goTime,100)
