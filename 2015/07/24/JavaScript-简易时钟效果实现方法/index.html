JavaScript 简易时钟效果实现方法

Posted on 2015-07-24   |   In JavaScript / JQuery

学习笔记：

Date() 时间对象，时 getHours() 分 getMinutes() 秒 getSeconds()

设置时间间隔 setInterval(function,1000) 每隔一秒执行一次函数，相似的还有 setTimeout(function,1000) 在1秒后执行函数，只执行一次。

如果需要关闭 setInterval() 可使用 clearInterval()

JavaScript 代码：

//给每个<10的数字前补0
function zero (n) {
    if (n<10) {
        return '0' + n;
    } else{
        return '' + n;
    };
}

window.onload = function (){
    function clock(){
        var oImg = document.getElementsByTagName('img');
        var oDate = new Date();
        var str = zero(oDate.getHours()) + zero(oDate.getMinutes()) + zero(oDate.getSeconds())  //获取时，分，秒后拼接成这样的字符串 var str = '015623'
        for (var i = 0; i < oImg.length; i++) {
                    oImg[i].src='images/'+str[i]+'.ico';
        };
    }
    setInterval(clock,1000)
    clock();  //在setInterval执行前运行一次，避免刷新归0
}
