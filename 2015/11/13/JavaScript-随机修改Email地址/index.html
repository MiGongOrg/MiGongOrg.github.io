JavaScript 随机修改 Email 地址

Posted on 2015-11-13   |   In JavaScript / JQuery

今天无意中发现一个轻论坛（Flarum http://flarum.org.cn/），简洁优雅的风格是我喜欢的调性。在开发者社区注册了一个账号，几分钟体验下来感觉还不错。但发现用户可以无限修改 Email，系统并会向修改后的新 Email 发送确认邮件，所以脑洞大开有了以下代码。

在浏览器控制台运行以下 JavaScript Code 会发生什么？（需修改注释部分代码）

JavaScript Code:

function Code(){
    var code = "";
    var codeLength = 9;
    var oNumber = new Array(1,2,3,4,5,6,7,8,9);
    var oMail = new Array("@qq.com","@yahoo.com","@foxmail.com","@gmail.com","@163.com","@126.com","@hotmail.com","@sina.com","@sohu.com");

    for(var i = 0; i < codeLength; i++) {
        var charIndex = Math.floor(Math.random()*9);
        code += oNumber[charIndex];
    }

    if(code.length != codeLength){
        Code();
    } else{
        code += oMail[charIndex];
    }
    showChange(code);
}

function showChange(a){

    // 修改按钮.click();

    var oInput = document.getElementsByTagName('input')[0];
    oInput.value = a;

    // 确认修改.click();
}

setInterval(Code, Math.random()*500+500 );
