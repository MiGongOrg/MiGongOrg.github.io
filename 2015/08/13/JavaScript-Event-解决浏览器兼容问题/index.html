JavaScript Event 解决浏览器兼容问题

Posted on 2015-08-13   |   In JavaScript / JQuery

在给“小奶瓶”聊天机器人添加键盘监听 ENTER 即可发送消息功能时，因火狐Firefox浏览器与低版本谷歌Chrome浏览器不支持 JavaScript Event 所以需要找到一个解决兼容问题的方法。

JavaScript Code：

//键盘监听
var oInputText = document.getElementById('inputText');
oInputText.onkeydown = function (ev){
    var oEvent = ev || event;
    if (oEvent.keyCode == 13) { //按下ENTER发送消息
        sendMessage();
    }
}

在上面的 JavaScript Code 中 ev 是事件处理函数的参数，该参数是事件对象，是由系统传递的。
