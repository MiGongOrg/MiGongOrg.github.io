CSS3 animation-delay 延迟执行 3D 动画

Posted on 2015-10-14   |   In HTML / CSS

核心 CSS3 Code：给 li 定义 translateZ 3D 转换，以及改变 li 伪元素 :after 和 :before 的 rotateX 与 translateY ，并添加 使用:nth-child() 选择器让每个 li 延迟执行动画 animation-delay

HTML Code：



*********

CSS Code：

*{
    margin: 0;
    padding: 0;
}
.container{
    position: absolute;
    top: 50%;
    left: 50%;
    /*border: 1px solid #777;*/
    margin: -75px 0 0 -75px;
    width: 150px;
    transform: rotateX(45deg) rotateZ(45deg);
    -webkit-transform: rotateX(45deg) rotateZ(45deg);
    transform-style: preserve-3d;
    -webkit-transform-style: preserve-3d;
}
ul{
    transform-style: preserve-3d;
    -webkit-transform-style: preserve-3d;
}
ul>li{
    list-style: none;
    display: inline-block;
    width: 50px;
    height: 50px;
    background-color: tomato;
    position: relative;
    float: left;
    transform: translateZ(50px);
    -webkit-transform: translateZ(50px);
    transform-style: preserve-3d;
    -webkit-transform-style: preserve-3d;
    animation: anim 1s infinite alternate;
    -webkit-animation: anim 1s infinite alternate;
    box-shadow: 200px 200px 10px 0px #777;
}
ul>li:after{
    content: "";
    width: 50px;
    height: 50px;
    background-color: tomato;
    display: inline-block;
    position: absolute;
    top: 0;
    left: 0;
    transform: rotateX(-90deg) translateY(50px);
    -webkit-transform: rotateX(-90deg) translateY(50px);
    transform-origin: 100% 100%;
    -webkit-transform-origin: 100% 100%;
}
ul>li:before{
    content: "";
    width: 50px;
    height: 50px;
    background-color: tomato;
    display: inline-block;
    position: absolute;
    top: 0;
    left: 0;
    transform: rotateY(90deg) translateX(50px);
    -webkit-transform: rotateY(90deg) translateX(50px);
    transform-origin: 100% 0;
    -webkit-transform-origin: 100% 0;
}
@-webkit-keyframes anim {
    from {
        -webkit-transform: translateZ(50px);
    }
    to {
        -webkit-transform: translateZ(5px);
    }
}
@keyframes anim {
    from {
        transform: translateZ(50px);
    }
    to {
        transform: translateZ(5px);
    }
}

/* 延迟执行动画 animation-delay */

ul>li:nth-child(1){
    animation-delay: 0.05s;
    -webkit-animation-delay: 0.05s;
}
ul>li:nth-child(2){
    animation-delay: 0.1s;
    -webkit-animation-delay: 0.1s;
}
ul>li:nth-child(3){
    animation-delay: 0.15s;
    -webkit-animation-delay: 0.15s;
}
ul>li:nth-child(4){
    animation-delay: 0.2s;
    -webkit-animation-delay: 0.2s;
}
ul>li:nth-child(5){
    animation-delay: 0.25s;
    -webkit-animation-delay: 0.25s;
}
ul>li:nth-child(6){
    animation-delay: 0.3s;
    -webkit-animation-delay: 0.3s;
}
ul>li:nth-child(7){
    animation-delay: 0.35s;
    -webkit-animation-delay: 0.35s;
}
ul>li:nth-child(8){
    animation-delay: 0.4s;
    -webkit-animation-delay: 0.4s;
}
ul>li:nth-child(9){
    animation-delay: 0.45s;
    -webkit-animation-delay: 0.45s;
}
