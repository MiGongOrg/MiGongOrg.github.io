CSS 实现《星辰坠落》

Posted on 2015-06-27   |   In HTML / CSS

在365awesomedesigners.com看到这幅画，于是乎就产生了一个想法，用CSS将它实现，并且增加一些动态效果。

效果的实现并不复杂，但如果想要追求完美细节，还是需要一点时间去完善的。经测试兼容：Firefox Opera Chrome ，其它浏览器未测试。

背景：

background: -webkit-gradient(linear, 100% 0, 0 100%, from(#201E31), to(#464465));
background: -webkit-linear-gradient(top right, #201E31, #464465);
background: -moz-linear-gradient(top right, #201E31, #464465);
background: -o-linear-gradient(top right, #201E31, #464465);
background: linear-gradient(top right, #201E31, #464465);

悬崖:

/* 悬崖 */
.rock{
    position: absolute;
    right: 0%;
    bottom: 0%;
    border-top: 0px solid transparent;
    border-bottom: 200px solid transparent;
    border-right: 300px solid #322E54;
    display: inline-block;
}

.rock::before{
    content: "";
    position: relative;
    right: 0%;
    bottom: 0%;
    border-top: 0px solid transparent;
    border-bottom: 100px solid transparent;
    border-right: 60px solid #322E54;
    display: inline-block;
}

.rock::after{
    content: "";
    position: absolute;
    right: -500%;
    top: 0%;
    border-top: 0px solid transparent;
    border-bottom: 30px solid transparent;
    border-right: 334px solid #504660;
    display: inline-block;
}

星球：

/* 星球 */

.star{
    width: 300px;
    height: 300px;
    margin: 0 auto;
    border-style: solid solid none none;
    border-color: #3D3848;
    border-width: 20px;
    border-bottom-right-radius: 50%;
    border-top-right-radius: 50%;
    border-top-left-radius: 50%;
    position: absolute;
    top: 30%;
    left: 50%;
    margin-top: -150px;
    margin-left: -150px;
    opacity: 0.3;
    box-shadow: 6px -6px 6px 0px #fff;
}
