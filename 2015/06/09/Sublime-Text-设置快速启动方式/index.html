Sublime Text 设置快速启动方式

Posted on 2015-06-09   |   In Sublime Text

  1. 将桌面 Sublime Text 快捷方式重命名为 sub
  2. 复制重命名后的 Sublime Text 快捷方
  3. 在F盘(任意盘)新建文件夹 sub（任意文件夹名） ,将重命名后的 Sublime Text 快捷方 sub 粘贴到该文件夹中
  4. 复制该文件夹路径，F:\sub
  5. 右键我的电脑->属性->高级系统设置->环境变量
  6. 编辑变量PATH，在变量值的末尾添加“;”分号，粘贴路径：F:\sub 后保存

    运行快捷键 Win+R 输入sub Enter后即可打开 Sublime Text

其它已安装软件设置快速启动方式步骤相同，系统默认常用快速启动方式还有：（个人）

notepad - 打开记事本
calc - 打开计算器

如需了解更多，请在搜索引擎查找关键词：运行命令大全
