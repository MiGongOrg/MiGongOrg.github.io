Posted on 2015-06-06   |   In JavaScript / JQuery

在本次练习中所用到的CSS知识点：

使用伪元素:before绘制三角形的方法：


content:””;

jQuery toggle()方法的使用：

首先引入JQ库，在练习中用到了360提供的CDN前端公共库

http://ajax.useso.com/ajax/libs/jquery/1.7.2/jquery.min.js?ver=3.4.2

使用toggle()方法：


$(‘#lon’).click(function(){
    $(‘#lonin’).toggle();
})

解释：当用户单击 ID 为 lon 的元素时，执行 function 中 ID 为 lonin 的元素 使用 toggle() 方法。
