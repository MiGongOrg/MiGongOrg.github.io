CSS实现全屏滚动视觉背景反差特效

Posted on 2015-06-08   |   In HTML / CSS

这是一种看似非常高大上的视觉背景反差特效，在很多知名网站上都有应用过。第一次看到时也感觉很神奇，视觉冲击效果很强，但实现方法要比我们想象中简单许多。

首先解决背景图片全屏问题：

html,body{height:100%}

实现这种视觉背景反差特效的主要CSS代码：

background-attachment: fixed;
