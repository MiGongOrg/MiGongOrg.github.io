纯CSS 绘制高大上 Login 登录页面 - 练习记录

Posted on 2015-06-05   |   In HTML / CSS

除背景图片以外，其它部分均用CSS绘制，主要学习与收获部分：
1.透明边框，rgba 的使用：

border: 5px solid rgba(0, 0, 0, 0.1);

2.伪元素 :before 和 :after 的使用：（绘制可爱脸型LOGO）主要应用边框圆角:

border-radius: 50%;

3.hover 鼠标移动显示阴影效果 box-shadow :

box-shadow: 0 1px 11px 0 #ccc,0 1px 11px 0 #ccc inset;

登陆框的整体效果实现还是挺简单的，主要应用了一些CSS3的特性，立体的登陆框和CSS绘制的LOGO图标是这个登陆页面的主要亮点。
