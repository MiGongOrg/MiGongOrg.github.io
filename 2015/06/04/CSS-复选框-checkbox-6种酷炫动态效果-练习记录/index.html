CSS 复选框（checkbox）6种酷炫动态效果 - 练习记录

Posted on 2015-06-04   |   In HTML / CSS

主要应用到的元素：

  • :after 伪元素的使用
  • box-shadow 为按钮增加立体感效果
  • opacity 元素透明，点击后显示与隐藏
  • cursor 鼠标指针效果 （pointer）
  • transition 过渡动态效果
  • transform 旋转效果 rotate(-40deg)
  • background 的线性渐变效果
background: linear-gradient(#FCFFF4 0%,#DFE5D7 40%,#B3BEAD 100%);
background: -moz-linear-gradient(#FCFFF4 0%,#DFE5D7 40%,#B3BEAD 100%);
background: -o-linear-gradient(#FCFFF4 0%,#DFE5D7 40%,#B3BEAD 100%);
background: -webkit-linear-gradient(#FCFFF4 0%,#DFE5D7 40%,#B3BEAD 100%);

input控制元素*

input[type=checkbox]:checked + label:after{opacity: 1;}
input[type=checkbox]:checked + label:after{left: 1;}
